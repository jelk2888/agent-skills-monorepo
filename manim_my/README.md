# Manim 动画创作技能（3b1b 风格增强版）

> Manim 物理/数学动画创作技能，内置3b1b标准色彩系统、5大心智模型、完整制作SOP、隐喻库和质量检核清单。

## 版本

v2.5.0

## 功能特性

- 🔬 **研究员模式**：先从网上调研相关文章、定理、实验和经典例题，提炼信息脉络，再制作深度教学视频
- 📷 **图片讲解视频**：从上传题目图片自动分析→题型识别→图形重建→讲解脚本→Manim代码生成
- 🎨 **3b1b风格DNA**：语义色彩系统、5大心智模型、动画节奏参数
- 📚 **150+官方动画示例**：创建/淡入/生长/指示/变换/文本/图论/表格/3D/特效
- 🧮 **多领域覆盖**：物理/数学/ML/范畴论
- 🔧 **完整工具链**：渲染管线/环境检查/项目模板

## 安装

将 `manim_my` 目录放置在技能目录下即可使用。

## 使用

在 Claude Code 中直接描述你想要的动画，技能会自动激活。
